중증도 분류(Triage)

중증도 분류(Triage)는 환자 상태의 위중함에 따라 처치 우선순위를 정하는 방법입니다.
뷰라보에서는 'START' 분류 방법으로 환자의 처치 우선 순위를 정하는 과정을 학습할 수 있습니다.

뷰라보의 중증도 분류 교육은
많은 인원도, 장비도 필요 없습니다.

시나리오와 무제한에 가까운 가상환자, 그리고 평가까지 뷰라보가 준비했습니다.
당신은 환자를 빠르고 명확하게 분류하고 우선순위를 정해보세요.

  • 시나리오 선택

    학습하고 싶은 시나리오를 선택해보세요.

  • 가상환자 생성

    시나리오에 맞게 가상환자가 생성됩니다.

  • 환자분류

    환자의 상태를 평가하고 분류해보세요.

  • 분류평가

    분류가 정확했는지 놓친 환자는 없는지 평가를 통해 확인해보세요.

뷰라보에서는 중증도 분류 방법 중 START 분류를 제공하고 있습니다.

뷰라보 중증도 분류 환자 평가

뷰라보에서는 환자의 상태를 평가할 수 있도록 기능을 구현했습니다.
START 분류 프로세스를 적용하였고, 세브란스 병원의 자문을 받아 콘텐츠를 제작했습니다.

그럼 분류 정확도를 확인해보세요.

  • 긴급한 환자부터 분류했는가?
  • 환자를 정확하게 분류했는가?
